# Mobile Owner Experience V1

Final Mile Margin mobile owner mode has been rebuilt around money, risk, and action.

## Included

- Command Dashboard
- Claims Center
- Receipt Approval Workflow
- New Bottom Navigation
- Team Performance View

## Owner Workflow

Owners now see what needs attention first, how much money is involved, and the next action to take. The mobile experience prioritizes profit, claims exposure, receipt approvals, team readiness, and recent activity instead of presenting raw forms.
